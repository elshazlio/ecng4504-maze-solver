> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make true left Ts visible when the sensor sequence is `00100 -> 01100 -> 00100` and `s0` never fires.

**Architecture:** Keep the controller self-contained in `working_code.ino`, preserve the current `s0Recently` and post-left-assist gates, and add one small piece of previous-sample state so `01100` can count as a left junction immediately after a clean centered approach. **Tech Stack:** Arduino-style C++ in a single `.ino` file with host-side C++ regression tests.
